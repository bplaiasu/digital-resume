# Digital Resume

A responsive digital resume built with **HTML5** and **CSS3** that showcases my professional experience as a **Senior Data Engineer**.

The project serves as a single source of truth for my resume, allowing me to maintain one version that can be viewed online, version-controlled with Git, and exported as a printable PDF.

## Running Locally

Clone the repository.

```bash
git clone https://github.com/bplaiasu/digital-resume.git
```

Navigate to the project.

```bash
cd digital-resume
```

Open the resume.

```bash
index.html
```

Or simply open `index.html` in your preferred browser.

---

## Exporting to PDF

The HTML version is the primary source.

To generate a PDF:

1. Open `index.html` in Chrome or Microsoft Edge.
2. Press **Ctrl + P**.
3. Choose **Save as PDF**.
4. Enable **Background graphics** (optional).
5. Save the generated PDF.

This ensures the PDF always matches the latest version of the resume.

## Why HTML Instead of Word?

Using HTML provides several advantages over traditional document editors:

- Full version control with Git
- Easy deployment using GitHub Pages
- Responsive layout
- Reusable styling
- Better long-term maintainability
- Fast updates
- Consistent online and PDF versions

Treating a resume as code makes it easier to evolve over time while preserving a complete history of changes.
